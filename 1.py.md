# Ultrasonic Sensor

## Wiring

| Distance Sensor Pin | Raspberry Pi Pin |
|------------|-----------------|
| VCC (5V)   | 5V (Pin 2)      |
| GND        | GND (Pin 6)     |
| TRIG       | GPIO 23 (Pin 16) |
| ECHO       | GPIO 24 (Pin 18) (⚠️ Needs both resistor! ⚠️) |
